E-commerce App

# Getting Started


## Step 1: Install Dependencies

```bash
# using npm
npm install

# OR using Yarn
yarn install
```


## Setp 2: Start Metro

```bash
# using npm
npm start

# OR using Yarn
yarn start
```


## Step 3: Start your Application

### For Android

```bash
# using npm
npm run android

# OR using Yarn
yarn android
```

### For iOS

#### Notes

> - Before building the app on iOS, you need to install CocoaPods inside the iOS folder.

---

```bash
# install pods
cd ios
pod install
cd ..

# using npm
npm run ios

# OR using Yarn
yarn ios
```
